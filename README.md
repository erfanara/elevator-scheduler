# elevator-scheduler
Elevator scheduler considering depreciation (direction change in elevator moves) and waiting time.

## Features
- Simplicity
- Flexible scheduling algorithm by calculating costs
- Scalable (You can increase number of elevators)
- Simple tkinter gui
- ...
